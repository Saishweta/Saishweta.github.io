Project Overview

Motivation

The intent of selecting this project is because of its practicality and application. In a real world scenario, we believe this could be used by common users to get an idea of different cuisines based on a set of ingredients.

Objectives

Through this project, we sought to answer the following questions:
1. How different cuisines may be identified based on certain combination of ingredients?
2. What are the most commonly used ingredients in a cuisine?
3. What is the distribution of ingredients in different countries?

Approach

We began our study by exploring the data set from Kaggle website detailing the ingredients and the corresponding cuisines. A series of exploratory analysis was performed with the hope of understanding the association of the variable with one another in the given data set. We used various modeling techniques to identify the best result. During the exploration phase, it was identified that three algorithms may be used to come up with the predictions: One using Random Forest classification, another using Decision Tree classification and third using Support Vector Machine (SVM).

Modelling

Analysis

The data set used for analysis (also known as a train data set) contains - Cuisines and the associated list of ingredients. The data set used for prediction (also known as test data set) contains the ingredients list on which the prediction is done in order to come up with the most accurate cuisine for that list.

The analysis and modelling was done on these data using three different classifiers namely:

  1. Decision Tree
  2. Random Forest
  3. Support Vector Machine (SVM)

During the analysis, we identified the performance and accuracy score of individual classifiers which is depicted in graph below.

Voila! Found the best classifier!! - SVM it is..

The classifier that gave 76% accuracy that too at a very less time (indicating high performance) is SVM, which is the best out of the three. Hence we chose SVM to do our prediction.
