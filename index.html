Cook It Up With The Fantastic Four

Project Overview

Motivation

The intent of selecting this project is because of its practicality and application. In a real world scenario, we believe this could be used by common users to get an idea of different cuisines based on a set of ingredients.

Objectives

Through this project, we sought to answer the following questions: 1. How different cuisines may be identified based on certain combination of ingredients? 2. What is the most commonly used ingredient in a cuisine? 3. What is the distribution of ingredients in different countries?

Approach

We began our study by exploring the data set from Kaggle website detailing the ingredients and the corresponding cuisines. A series of exploratory analysis was performed with the hope of understanding the association of the variable with one another in the given data set. We used various modeling techniques to identify the best result. During the exploration phase, it was identified that three algorithms may be used to come up with the predictions: One using Random Forest classification, another using Decision Tree classification and third using Support Vector Machine (SVM).
